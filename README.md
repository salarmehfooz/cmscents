# 🌿 CM Scents

**CM Scents** is a modern, elegant fragrance brand website designed to showcase premium perfumes with a luxurious and minimal digital experience.

---

## ✨ Overview

CM Scents is built to represent a fragrance brand with a strong visual identity, smooth user experience, and clean product presentation.

This project focuses on:

- Luxury-inspired UI/UX design
- Responsive layout for all devices
- Smooth navigation and structure
- Fast and optimized frontend performance

---

## 🚀 Features

- 🌸 Modern fragrance brand design
- 📱 Fully responsive (mobile, tablet, desktop)
- 🖼️ Clean product showcase layout
- ⚡ Fast and lightweight performance
- 🎨 Minimal and luxury-inspired UI
- 🔧 Easy to customize and scale

---

## 🛠️ Tech Stack

- **Frontend:** React+Vite
- **Styling:** Tailwind CSS
- **Language:** JavaScript
- **Build Tool:** Vite
